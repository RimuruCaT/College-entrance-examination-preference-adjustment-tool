# 高考志愿顺序调整工具

一个纯前端小工具，用于在浏览器中调整 CSV 志愿表顺序。所有数据都在本地浏览器解析和导出，不会上传到服务器。

## 功能特点

- 上传 CSV 后自动识别表头和志愿记录。
- `25年位次` 列为 `不参与排序` 的专业保持原始相对顺序，并固定置顶。
- 其余志愿可通过拖拽或上下按钮调整顺序。
- 可设置“最大建议移动幅度”，当某个志愿移动过大时给出醒目提醒，帮助用户避免过度打乱原有安全顺序。
- 导出 CSV 保持导入时的列格式，并按最终顺序更新 `序号` 列。
- 支持含英文逗号、双引号、换行等标准 CSV 转义内容。

## 使用方法

1. 直接用浏览器打开 `index.html`。
2. 点击“选择 CSV 文件”上传志愿表。
3. 固定置顶区无需操作；在“可调整志愿”区拖拽卡片或点击上下箭头。
4. 根据移动提醒检查是否过度调整。
5. 点击“导出调整后 CSV”。

## CSV 示例

```csv
序号,院校代码,院校名称,专业代码,专业名称,25年位次
1,1,浙江大学,19,工科试验班(计算机),不参与排序
2,1,浙江大学,26,工科试验班(低空科技创新班),不参与排序
3,1,浙江大学,39,"工科试验班(海洋,含智慧海洋卓越班)",不参与排序
4,2,复旦大学,01,计算机科学与技术,1200
5,3,上海交通大学,02,电子信息类,1350
```
